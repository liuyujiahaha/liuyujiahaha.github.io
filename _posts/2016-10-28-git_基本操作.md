### Git 基本操作

1. git clone [链接地址] ：从GitHub克隆项目；
2. git status ：查看本地文件状态；
3. git add . ：添加文件到缓存，该.符号可以修改为特定的文件名;
4. git commit -m "[修改内容]" ：提交至本地；
5. git push ：上传更新至服务器；
6. git pull ：更新服务器内容至本地；

* 删除文件：rm 文件名 —— git rm 文件名 —— git commit -m "[修改内容]"
